# House Price Prediction

End-to-end ML project using Python.

### Input Features
- Posted By
- Under Construction
- RERA Registration
- BHK Number
- BHK / RK
- Square Feet
- Ready to Move
- Resale
- Longitude
- Latitude

## Workflow
- ML training in Jupyter Notebook
- Model serialized using pickle
- Flask UI for user input
- CI/CD with GitHub Actions
- Deployment on AWS EC2

## Tech Stack
- Python
- Scikit-learn
- Flask
- GitHub Actions
- AWS EC2
